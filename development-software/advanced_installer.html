Advanced Installer

Advanced installer integrated with visual studio and creates installer packages for windows. Advanced installer has a premium version therefore some features are restricted from the free version. The free version still can make MSI installer packages in one file and includes Visual Studio integration.




System Requirements
  • Processor: Core 2 class CPU
  • Memory: 1 GB RAM
  • Graphics: 1366x768 screen resolution
  • Storage: 2 GB hard drive space
