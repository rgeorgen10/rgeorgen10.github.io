Ubuntu Server

Ubuntu server is an operating system based on the linux kernel made by Canonical intended for server usage. Ubuntu server is a much more modular OS than Ubuntu Desktop. Ubuntu server can be great for more advanced users wanting less bloat but still wanting a Ubuntu base.




System Requirements
  • RAM: 1 GB
  • Processor: 1 GHZ or Faster
