Rocky Linux

Rocky Linux is based on Red Hat Enterprise Linux and is very similar to CentOS. Rocky Linux is mostly meant for servers or users who need a very stable operating system. Like Red Hat, Rocky Linux almost never breaks and can run on a system for years without even needing to be restarted.




System Requirements
  • RAM: 2 GB
  • Storage: 20 GB
  • Bootable Media
