Software Name



7-Zip

7-Zip is a free and open source file extracter for Windows. 7-zip works with many different types of archives including zip files and rar files. 7-zip also can compress the size of files which can be useful for emailing files.




System Requirements
  • Operating Systems: Windows XP-10, Windows Server 2000-2019
