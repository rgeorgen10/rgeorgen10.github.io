Vivaldi

The Vivaldi Web Browser is an open-source web browser based on the chromium open-source project similarly to Google Chrome. The Vivaldi web browser supports a large amount customization allowing a user to change the color, and location of everything on the screen.




System Requirements
  • Operating System: Windows 7 / 8 / 10
  • Memory: 512 MB
