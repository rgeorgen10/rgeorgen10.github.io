Lutris

Lutris is an open-source game client for linux. The goal of steam is to make windows only games play on Linux without a lot of advanced configuration. Games from Steam, Battle.NET, COG, Origin and many other sources work. Some Windows only games actually get better FPS on Linux because there is less overhead in Linux than there is in Windows.




System Requirements
  • Operating Systems: Debian Based Systems, RHEL Based Systems, Arch Based Systems and many other linux distros.
