EagleGet

EagleGet download manager is a free download manager available for Microsoft Windows. Features include browser integration, scheduler, and FTP support. The EagleGet installer is virus free but does contain bundle-ware so be aware during installation.




System Requirements
  • Operating System: Windows XP / Vista / 7 / 8 / 10
  • Memory: 64 MB
  • Storage: 6 MB
