MiniTool Partition Manager

MiniTool Partition Wizard is a free partition manager for Windows that supports a variety of different file systems and can do actions that the built in windows partition manager can't do. This software can shrink, extend, delete, and create partitions in many different file systems.




System Requirements
  • Operating System: Windows 7 / 8 / 10
